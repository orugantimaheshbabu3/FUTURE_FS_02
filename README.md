# 🚀 Mini CRM – Client Lead Management System

A full-stack web application designed to help businesses manage client leads efficiently. This Mini CRM allows admins to track incoming leads, update lead statuses, add follow-up notes, and manage customer interactions from a secure dashboard.

---

# 📌 Project Overview

This project was built as part of a Full Stack Web Development Internship Task.

The CRM system helps businesses:

- Store leads from contact forms
- Track lead progress
- Manage follow-ups
- Convert leads into clients

This project simulates a real-world client management workflow used by agencies, freelancers, and startups.

---

# ✨ Features

## ✅ Lead Management
- Add new leads
- View all leads
- Edit lead details
- Delete leads

## ✅ Lead Status Tracking
Track lead progress with statuses:
- New
- Contacted
- Converted

## ✅ Follow-Up Notes
- Add notes for each lead
- Maintain communication history

## ✅ Secure Admin Access
- Admin login authentication
- Protected dashboard routes

## ✅ Dashboard
- Clean and responsive UI
- Organized lead table
- Easy management system
